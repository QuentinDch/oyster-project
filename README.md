# Oyster Project | Website

## Description

This project showcases a minimalist and sleek website for a fictional restaurant, built with HTML, SASS, React, and utilizing GSAP for smooth and precise animations. The goal is to offer an enjoyable and modern user experience, with a simple yet elegant design, perfectly suited for users looking for smooth and intuitive navigation.

## Technologies Used

- **HTML** : Page structure.
- **CSS (SASS)** : Page styling with a minimalist design.
- **React** : Managing the user interface with reactive components, ensuring a smooth and interactive experience.
- **GSAP (GreenSock Animation Platform)** : Smooth and precise animations, creating dynamic and engaging transitions.

## Features

- **Clean Homepage** : A minimalist homepage showcasing the restaurant with high-quality visuals and smooth animations.
- **Reservations** : A reservation form allowing users to contact the restaurant.
- **Transition Animations** : Smooth animations between pages and on hover for dynamic interaction.
- **Responsive Design** : The site is adaptable for optimal use on all devices, from desktop to mobile.

## Performance and Accessibility

- Performance: Optimization of images and resources to ensure fast loading times, particularly through the management of GSAP animations to avoid overload.
- Accessibility: Use of HTML5 semantic tags and accessibility attributes to ensure the site is accessible to all users, including those with disabilities.
